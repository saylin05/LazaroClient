# LAZARO-CLIENT

Aplicación móvil para la gestión y administración de clientes del sector textil.

## Descripción

LAZARO-CLIENT es una solución tecnológica diseñada para modernizar la gestión comercial de empresas textiles. Permite organizar, almacenar y administrar la información de clientes de manera centralizada, reemplazando el manejo manual y descentralizado de registros.

## Funcionalidades

- **Registro de clientes** – Datos personales y comerciales en un solo lugar.
- **Clasificación por zonas geográficas** – Localización y segmentación de clientes por región.
- **Envío de promociones** – Difusión de ofertas, precios actualizados y novedades.
- **Catálogos digitales** – Compartir muestrarios y catálogos directamente desde la app.
- **Base de datos segura** – Almacenamiento centralizado con fácil acceso y sin riesgo de pérdida de información.

## Objetivo

Facilitar el seguimiento comercial, la clasificación por zonas y el envío de información promocional a los clientes de una empresa distribuidora de telas y productos textiles en Costa Rica.

## Público objetivo

- Microempresas de textiles
- Negocios de venta de telas
- Distribuidores de productos textiles
